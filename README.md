# Inclusive Study Companion 🎓✨

## 📌 Overview
Inclusive Study Companion is an AI-powered educational assistant designed to help students with personalized learning support.

## 🚀 Features
- AI-based study assistance
- User-friendly interface
- Accessible learning support
- Smart responses

## 🛠 Tech Stack
- Python
- Flask
- HTML
- CSS
- JavaScript

## 📂 Project Structure
- app.py → Main backend file
- templates/ → HTML files
- static/ → CSS & JS files
- requirements.txt → Dependencies

## ▶️ How to Run Locally

1. Clone the repository
2. Install requirements:
   pip install -r requirements.txt
3. Run:
   python app.py
